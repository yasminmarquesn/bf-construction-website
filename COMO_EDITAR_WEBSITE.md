# 📝 Como Editar o Seu Website

Existem **3 formas** de fazer alterações no seu website BF Construction. Escolha a que melhor se adequa a si!

---

## 🎯 Opção 1: Editar Online (MAIS FÁCIL - Recomendado)

**Sem precisar de instalar nada! Edita directamente no navegador.**

### Passos:

1. **Aceda ao GitHub:**
   - https://github.com/yasminmarquesn/bf-construction-website

2. **Clique em "src"**

3. **Clique em "App.jsx"** (ficheiro principal)

4. **Clique no ícone de lápis** (Edit) no canto superior direito

5. **Faça as alterações** que quer

6. **Clique em "Commit changes"** no canto inferior direito

7. **Escreva uma mensagem** (ex: "Actualizar telefone de contacto")

8. **Clique em "Commit changes"**

9. **Aguarde 2-5 minutos** - O website actualiza automaticamente!

## 🔧 Opção 2: Editar no Computador (Mais Controlo)

**Se prefere trabalhar no seu computador e depois enviar para o website.**

### Requisitos:
- Git instalado (https://git-scm.com/)
- Visual Studio Code (https://code.visualstudio.com/) - GRÁTIS

### Passos:

1. **Abra o Terminal/Prompt de Comando**

2. **Clone o repositório:**
```
git clone https://github.com/yasminmarquesn/bf-construction-website.git
```

3. **Entre na pasta:**
```
cd bf-construction-website
```

4. **Abra no Visual Studio Code:**
```
code .
```

5. **Edite os ficheiros** que quer

6. **Guarde as alterações** (Ctrl+S)

7. **Volte ao Terminal e execute:**
```
git add .
git commit -m "Descrição das alterações"
git push
```

8. **Aguarde 2-5 minutos** - O website actualiza automaticamente!

## ⚠️ Dicas Importantes:

✅ **Sempre faça backup** antes de grandes alterações
✅ **Teste as alterações** no navegador após 5 minutos
✅ **Não delete ficheiros importantes** (App.jsx, App.css, etc.)
✅ **Se cometer um erro**, pode reverter as alterações no GitHub

## 🆘 Se Algo Correr Mal:

2. **Clique em "History"** (Histórico)
3. **Seleccione a versão anterior**
4. **Clique em "Revert"**
5. **Pronto! Website volta ao normal**
