## 🔗 Próximo Passo: Ligar o Domínio Permanente

Para ligar o domínio **bfconstruction.co.uk** ao website:

### PASSO 1: Registar o Domínio (se ainda não tem)
1. Aceda a: https://www.namecheap.com
2. Procure: `bfconstruction.co.uk`
3. Compre o domínio (~£10-15/ano)
4. Guarde as credenciais de acesso

### PASSO 2: Configurar DNS no Namecheap
1. Aceda ao Namecheap com as suas credenciais
2. Vá para "Manage" do domínio bfconstruction.co.uk
3. Clique em "Advanced DNS"
4. Adicione os registos DNS do Vercel:
   - **CNAME:** www → bf-construction-website.vercel.app
   - **A:** @ → 76.76.19.89

### PASSO 3: Adicionar Domínio no Vercel
1. Aceda a: https://vercel.com/dashboard
2. Clique no projecto "bf-construction-website"
3. Vá para "Settings" → "Domains"
4. Adicione o domínio: bfconstruction.co.uk
5. Siga as instruções para verificar

### PASSO 4: Aguardar Propagação DNS
- Pode levar 24-48 horas para o domínio estar completamente ativo
- Verifique em: https://www.whatsmydns.net
