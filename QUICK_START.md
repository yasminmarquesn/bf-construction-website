# BF Construction - Quick Start Guide

## 🚀 Próximos Passos para Publicar o Website

O website está **100% pronto** para publicação! Siga estes passos simples:

---

## PASSO 1: Registar Domínio (5 minutos)

**Escolha uma opção:**

### Namecheap (Recomendado)
- Aceda a: https://www.namecheap.com
- Procure: `bfconstruction.co.uk`
- Compre o domínio (~£10-15/ano)

### GoDaddy
- Aceda a: https://www.godaddy.com
- Procure: `bfconstruction.co.uk`
- Compre o domínio

**Guarde o email e password de acesso!**

---

## PASSO 2: Criar Conta GitHub (5 minutos)

1. Aceda a: https://github.com/signup
2. Crie uma conta com email e password
3. Guarde as credenciais

---

## PASSO 3: Fazer Upload do Código (10 minutos)

Siga as instruções em `GITHUB_SETUP.md`

**Resumo:**
1. Crie repositório em GitHub
2. Faça push do código
3. Pronto!

---

## PASSO 4: Configurar Vercel (5 minutos)

1. Aceda a: https://vercel.com
2. Clique em "Sign Up"
3. Escolha "Continue with GitHub"
4. Autorize o Vercel
5. Clique em "New Project"
6. Seleccione `bf-construction-website`
7. Clique em "Import"
8. Aguarde o deploy (2-3 minutos)

---

## PASSO 5: Apontar Domínio (10 minutos)

### No Vercel:
1. Vá a "Settings" > "Domains"
2. Clique em "Add Domain"
3. Digite: `bfconstruction.co.uk`
4. Escolha "Using Nameservers"
5. **Copie os nameservers fornecidos**

### No Registador de Domínio (Namecheap/GoDaddy):
1. Faça login na sua conta
2. Vá a "Nameservers"
3. Escolha "Custom DNS"
4. **Cole os nameservers do Vercel**
5. Clique em "Save"

---

## ⏳ Aguarde 24-48 Horas

As alterações de DNS levam tempo a propagar. Após 24-48 horas:

✅ Aceda a https://bfconstruction.co.uk
✅ O website deve estar ao vivo!
